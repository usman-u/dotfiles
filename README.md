# dotfiles

These are my linux dotfiles, for linux systems.

### How to install:
1. Clone the repository.
2. Run the `install.sh` script...easy.

### What get installed?
* Regular System updates
* Symbolic link to `.vimrc`
* Symbolic link to `.tmux.conf`
* Bash alias of `ls -al` to `ls`
* Ookla CLI Speedtest
* Vim
* Htop
* Tmux
* Nmap
* Curl
* Git
* Neofetch
* Finglet
* Lolcat
